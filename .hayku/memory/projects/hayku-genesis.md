# Hayku Agent 創建記錄

Hayku 是一個自主的企業軟體建構代理，具備持續記憶、工具自擴展、多模型支援。

## 核心設計決策

- **多模型架構**：不鎖定單一 LLM，根據任務選擇最適模型
- **工具自生成**：Agent 判斷任務需要時自行建立新工具，初期自行生成，日後考慮工具市場
- **安全分級**：Level 0-3 權限分級制度
- **記憶系統**：檔案系統為基礎，日後可擴展至向量 DB
- **部署彈性**：內部 Docker 或雲端皆可

## 起步策略

方案 A：由內而外 — 先建 Memory + Tool Manager 核心，再加 Planner / Executor

## 脈絡

使用者 Anthony 需要一個能整合一致性企業資訊的自主代理，重視效率和高度自主性。技術選擇不預設限制，由 Agent 根據需求決定。
